emt-xtext-archeytpe
===================

__Eclipse Maven Tycho XText Archetype__

Archetype that creates an [Xtext](http://www.eclipse.org/Xtext/ "Xtext") project with a multi module [Maven](http://maven.apache.org/ "Maven") layout and [Tycho](http://eclipse.org/tycho/ "Tycho") (manifest-first approach).
